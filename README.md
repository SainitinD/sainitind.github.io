#### Why did I make this website?
I made my old personal website but that was made following a tutorial and not something I can call my own. So, I set out to make a website that I developed and put in my learned best practices in software and full-stack development. Some things I thought about when building this include ...
* Dependency injection
* Loose Coupling
* High Cohesion
* CSS Variables, selectors, animations
* Responsiveness
* UI/UX etc
* Organization of files, components, assets etc

#### Tech Stack
* VueJS
* Typescript
* HTML
* CSS

#### Notable Learnings/Achievements
* Improved understanding of CSS and dependency injection in the frontend
* Improved understanding of UI/UX and responsive websites
